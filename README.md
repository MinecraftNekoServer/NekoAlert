# NekoAlert

NekoAlert 是一个简单的 Minecraft 服务器插件，用于定期向所有在线玩家发送自定义消息。

## 功能

- 定时向所有在线玩家发送消息
- 支持颜色代码和链接
- 可配置的消息间隔时间
- 提供指令重载配置文件

## 配置

配置文件位于 `config.yml`：

```yaml
# 消息发送间隔时间（秒）
interval: 60

# 消息列表，支持颜色代码和链接
messages:
  - "&a[公告] &f欢迎来到服务器！"
  - "&b[提示] &f请遵守服务器规则。"
  - "&c[警告] &f不要破坏他人建筑。"
  - "&e[活动] &f即将举办建筑比赛，敬请期待！"
```

## 指令

- `/nekoalert reload` - 重载配置文件

## 权限

- `nekoalert.reload` - 使用重载指令的权限
